# Mideliz 

Mideliz is a web-based application developed as part of a food entrepreneurship project.  
The main goal of this project is to solve real-world problems through software, focusing on clean structure, scalability, and maintainability.

## Project Overview

Mideliz aims to support a food service model based on catalog sales, offering balanced meals for work environments and events.  
From a technical perspective, the project is designed as a learning and experimentation space for applying software engineering principles in a real context.

## Motivation

This project was created to:
- Apply software development skills to a real business scenario
- Improve decision-making around software structure and scalability
- Connect technical development with entrepreneurial thinking
